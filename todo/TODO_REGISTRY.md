# TODO Registry

All future work items, ideas, and planned improvements for this project are tracked here. Each item may have a dedicated `.md` file in this directory with full details.

## Status Key

| Status | Meaning |
|--------|---------|
| `open` | Not yet started |
| `in-progress` | Actively being worked on |
| `blocked` | Waiting on something external |
| `complete` | Done |
| `wont-do` | Decided against |

## Priority Key

| Priority | Meaning |
|----------|---------|
| `critical` | Blocks progress or correctness |
| `high` | Important and should be done soon |
| `medium` | Valuable but not urgent |
| `low` | Nice to have |
| `idea` | Speculative, no commitment |
