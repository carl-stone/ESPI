## Status Key

| Status | Meaning |
|--------|---------|
| `open` | Not yet started |
| `in-progress` | Actively being worked on |

## Priority Key

| Priority | Meaning |
|----------|---------|
| `critical` | Blocks progress or correctness |
| `high` | Important and should be done soon |
| `medium` | Valuable but not urgent |
| `low` | Nice to have |
| `idea` | Speculative, no commitment |
